Galactic Frontier Project

合意事項

本プロジェクト「Galactic Frontier」において、参加者の皆様が制作される建築物、デザイン、プログラム、テキスト、画像、音声、その他一切の創作物（以下、「本著作物」と総称します）に関する著作権およびその他一切の知的財産権の取り扱いにつきましては、以下の通りとさせていただきます。

基本方針

プロジェクトの円滑な運営と、将来的な成果物の公開、発展を見据え、権利の帰属および利用方法については、参加者全員が以下の共通合意事項に基づくものとし、個別の協議による例外は設けません。これにより、権利管理の簡素化およびトラブルの未然防止を図ります。

合意内容

  • 1. 成果物の権利帰属：
    本プロジェクトで制作されたすべての成果物は、本プロジェクトの共同著作物として扱われます。これらの成果物を他のサーバーやプロジェクト等で使用する場合は、事前にプロジェクト管理者の許可を得る必要があります。
  • 2. SNS等での公開：
    成果物に関する画像・動画などをX（旧Twitter）やその他SNS・Webサイト等に投稿することは可能ですが、必ず「©2025 Tsubu Server All Rights Reserved.」というクレジットを、投稿文末または画像下部など明確に視認できる形で表記してください。
  • 3. プロジェクト辞退時の対応：
    プロジェクトからの辞退を希望される場合は、必ず事前にプロジェクト管理者へ連絡し、対応をお待ちください。
    万が一、10日以上連絡が取れない場合は、DiscordのDM等を通じてご連絡を差し上げます。それでも2週間以上連絡がない場合は、プロジェクトから辞退扱いとし、参加権限を解除いたします。

本プロジェクトへの参加を希望される方は、上記合意内容に同意いただいたものとみなします。ご不明な点がございましたら、応募時にその旨をお伝えいただくか、面談時にご質問ください。
